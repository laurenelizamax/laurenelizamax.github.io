Before writing a line of code, I spent years working with people in ways that required real listening — as a language educator, a caregiver, and a professional organizer. What I learned is that helping someone well means understanding what they actually need — and that instinct still shapes how I build software.

Since 2020 I've been an application developer at Prime Health Services, where I build React + TypeScript interfaces that internal teams use every day. I own UI/UX decisions end-to-end, design the C#/.NET Core APIs behind them, and keep releases reliable with CI/CD in Azure DevOps and a healthy test suite. I also mentor junior developers and review code across the team.

I'm most interested in work where thoughtful engineering meets real human impact — maintainable code, reliable releases, and tools that genuinely save people time. (Also: fluent in French, if your codebase needs it.)
